# Demo script (≈12–15 minutes)

**Persona:** Owner of **“Urban Glow Studio”** — unisex salon in **Patia, Bhubaneswar** (demo data in seed matches this story).

**Environment:** `INGESTION_MODE=mock`, seeded DB, dashboard at `:3000`, API at `:4000`.

---

## 1. Open (30 s)

> “We help local businesses grow on **Instagram** without a full marketing team. I’ll show you connect, numbers, and AI help — all in one place.”

## 2. Login (1 min)

- Open dashboard → **Login**  
- **Primary:** **`demo@demo.com` / `Demo1234!`** (operator flow). **Alternate (client-only):** **`salon@pilot.demo` / `pilot123`**  
- Land on **Analytics** — point to nav: **Insights**, **Leads**, **Onboarding**.

## 3. Connect Instagram (2 min)

- **Onboarding** → Connect Instagram (or explain OAuth + Meta app for production).  
- In mock mode: “Jobs run in the worker; seed data simulates posts already synced.”

## 4. Dashboard snapshot (2 min)

- **Analytics** — total posts, engagement, best hour, charts.  
- Tie to business: *“Evening slots match when people book after work.”*

## 5. Interpretation (2 min)

- Top posts vs weaker posts — *“Reuse hooks that worked; refresh underperformers.”*

## 6. AI insight (2 min)

- **Insights** → generate content-performance insight (or show pre-generated).  
- Say clearly: *“Suggestions are from **your** data plus AI wording — not magic.”*

## 7. Weekly focus (1 min)

- **Insights** → **This week’s focus** → **Get weekly focus** (stores a recommendation for the client).  
- One clear action line — tie to *“less guesswork this week.”*

## 8. Captions (2 min)

- Generate captions for a **weekend offer** or **festive package** — local tone.

## 9. Close — outcome (1 min)

> “Outcome we pilot for: **clearer posting rhythm + less guesswork** — we measure with you: posts per week, DMs, or bookings you already track.”

**Do not claim:** guaranteed reach, automated posting to IG without Meta approval, or fake client logos.
